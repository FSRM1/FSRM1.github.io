Empire_LupinOne

Empire: LupinOne

信息收集

首先扫描存活主机。

得到ip地址为192.168.29.170

然后扫描端口信息，进行服务的访问。

80和22端口开放，没啥特别的漏洞，那就去扫一下后台看是否存在敏感文件。

访问/manual 会跳转到一个Apache HTTP Server Version 2.4 服务的界面。

没找到什么可以利用的漏洞，然后去访问robots.txt

可以看到有一个路径 我们去访问一下

查看网页源代码

猜测还有其他的路径，只不过是没扫到。

网上搜了一下，利用ffuf对~后面的内容进行fuzz

可以看到secret,访问一下

很明显了，在这个目录下接着扫后台。

漏洞利用

去访问这个文件。

一段经过base加密过后的内容 ，经过测试发现是base58

得到了私钥，我们可以去ssh登录一下。

将私钥放在一个passwd.txt文件里面 然后利用ssh2john转换成john可以爆破的形式 然后进行爆破密码即可。

可以得到密码：P@55w0rd!

然后就是ssh远程登录进去。

注意需要修改私钥文件的权限 修改为600

提权

在home目录下 存在两个用户 分别是 icex64和arsene

sudo -l 看了一下 能发现一个python文件

看一下内容。

导入了webbrowser这个，然后我们利用find查找一下

查看一下webbrowser.py文件

导入了挺多模块的，然后我们在里面添加一个os.sysytem(“/bin/sh”)

使用vi进行文件的编译

sudo -l 看一下

发现可以进行pip提权

在这个网站上可以看到提权的信息

https://gtfobins.github.io/gtfobins/pip/#sudo

提权成功.
